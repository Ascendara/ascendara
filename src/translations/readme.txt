`en-orginial.json` is the orginial language file. All other language files are
 based on this file and are translated with Google Translate.
json-translator is used to translate the orginial language file to other 
languages. View it at https://github.com/mololab/json-translator




Text may not be 100% accurate.

If there is an error in a translation, please open an issue 
on GitHub (https://github.com/tagoWorks/ascendara).
