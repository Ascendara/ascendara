### Quick Start

> ‚ö†Ô∏è **Note:** Some API features like reporting and analytics services will not work on the public version of the app. Additionally, you will not be able to run games in development mode. Check the [Developer Docs](https://ascendara.app/docs/developer/build-from-source#important-limitations) for more information.

1. **Clone the Repository**
   ```sh
   git clone https://github.com/tagoWorks/ascendara.git
   ```

2. **Install Dependencies**
   ```sh
   npm install -r
   ```

3. **Build the App**
   ```sh
   npm run dist
   ```
